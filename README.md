# BookVault

A full-stack MERN application designed to demonstrate complete CRUD functionality. Users can easily add, view, edit, and delete book records through a clean and responsive interface.

---

## Live Demo

---

## Features

- Add, view, edit, and delete book records
- Modal popup for viewing book details
- Dark mode support
- Toast notifications for user feedback
- Form validation
- Responsive design for all screen sizes

---

## Tech Stack

**Frontend**

- React
- Context API (global state management)
- Tailwind CSS
- React Router DOM

**Backend**

- Node.js
- Express
- MongoDB & Mongoose
